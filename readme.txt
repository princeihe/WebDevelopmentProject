Deployment Instructions

To deploy this project, follow the steps below:

Make sure you have Node.js installed on your machine.
Open a terminal and navigate to the project directory.
Run the following command to start the application:
node src/index.js or nodemon src/index.js
Access the application in your web browser at http://localhost:3000.


API keys are required for accessing certain features of the project. The keys are already included in the files search.html and searchlogged.html. If you need a new API key, follow these steps:

Register for a basic account on RapidAPI.com.
Once registered, navigate to the API documentation to find the API key generation section.
Generate a new API key for your project.
Replace the existing API keys in the search.html and searchlogged.html files with the new key.
